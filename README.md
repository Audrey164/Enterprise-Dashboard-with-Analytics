# Enterprise Analytics Dashboard

An enterprise-grade dashboard with real-time analytics, KPIs, reporting tools, and admin management.

## Features
- Real-time KPIs  
- Charts (Bar, Line, Pie)  
- User roles (Admin / Manager)  
- Activity logs  
- Export (PDF/Excel)  

## Stack
- Vue.js + Tailwind  
- Laravel API  
- MySQL  
- Redis Cache  

## Setup
```bash
npm install && npm run build
composer install
php artisan migrate
